# Java User Registration and Login System

## Project Overview
This project is a simple Java console-based application that allows a user to:
- Register with a **username**, **password**, and **phone number**.  
- Log in with the registered username and password.  
- Validate login credentials against the stored registration details.  

It was developed as part of my coursework at **Rosebank College**.  

---

## Features
- Register with:
  - Username  
  - Password (with validation rules)  
  - Phone number (in +27 South African format)  
- Login functionality:
  - Checks if the entered username and password match the registered details.  
  - Displays a success or failure message accordingly.
